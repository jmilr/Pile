# Pile

A minimal MDX editor playground built with Next.js, TypeScript, Sass, and Pier.

## Getting started

```bash
npm install
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000) to start editing.

## Features

- Local front matter + body editing with live MDX serialization using `gray-matter`.
- Preview mode powered by `react-markdown`.
- Optional upload hook that injects asset links into the editor content.
- Pier-based utility styling with a responsive, mobile-first layout.
